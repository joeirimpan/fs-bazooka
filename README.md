# fs-basooka

Nukes directory / files randomly

## Usage
```
./fs-bazooka --path "/home/quarantine" --upshot 20
```

| Params |  Required / Optional  |
|--------|-----------------------|
|`path`  | required              |
|`upshot`| optional defaults to 6|

## Build

```
make build
```

## Run

```
make run
```
